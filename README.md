# Data-Science

## Student Result Management System

This program helps teachers:

- Add student records
- View all students
- Update scores
- Search students
- Calculate grades
- Display class statistics

## INVENTORY MANAGEMENT SYSTEM

This program helps staff members to:

- Store products and quantities.
- Add new products.
- Update product quantities.
- Search for products.
- Remove products.
- Display inventory records.
- Generate inventory statistics.
- Identify low-stock items.
